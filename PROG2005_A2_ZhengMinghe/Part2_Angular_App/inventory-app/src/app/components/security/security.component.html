Privacy and Security Analysis

Compliance with ULO4: Mobile & Web System Requirements

1. Data Integrity and Input Validation

In TechSync Pro, we prioritize data integrity by utilizing Angular's Reactive Forms. This approach allows for synchronous and asynchronous validation. Specifically:

  • Type Enforcement: We use regular expression patterns (Validators.pattern) to ensure that numeric fields like Price and Quantity do not accept alphabetic characters, preventing logic errors and potential injection vulnerabilities.
  • Sanitization: Angular automatically sanitizes values to prevent Cross-Site Scripting (XSS) attacks, ensuring that malicious scripts cannot be executed within the browser session.

2. Session-Based Data Persistence

Following the assessment brief, this application functions without a server-side backend. Data is stored within a Service-managed singleton.

Security Consideration: While this protects user data from being intercepted over a network, the data resides in the browser's memory. To enhance privacy, we implement clear state management where data is cleared once the browser session ends, adhering to the principle of Ephemeral Storage.

3. Privacy and The Principle of Least Privilege

In a mobile context, privacy is paramount. Our app respects user privacy by:

  • Minimal Data Collection: Only essential inventory fields are required. We do not request access to sensitive mobile hardware (Camera, GPS, or Contacts) as they are not required for core functionality.
  • Supplier Privacy: Sensitive supplier information and internal comments are kept local to the session, preventing accidental exposure to unauthorized third parties.

4. Future Security Recommendations

To transition this prototype into a production-ready mobile application, the following security measures would be implemented:

  • HTTPS/TLS: Encrypting data in transit to prevent Man-in-the-Middle (MitM) attacks.
  • JWT Authentication: Secure user login and authorization using JSON Web Tokens.
  • Encrypted Local Storage: Using SecureStorage plugins if the app is deployed on mobile platforms via Capacitor or Cordova.
